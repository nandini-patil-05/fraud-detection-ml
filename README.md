# Fraud Detection on Bank Payments Using Machine Learning

## Project Overview
This project focuses on detecting fraudulent bank transactions using Machine Learning techniques. With the rapid growth of digital payments, identifying fraudulent activities has become essential for ensuring financial security. The system analyzes transaction data and classifies them as fraudulent or legitimate based on learned patterns.

---

## Objectives
* Detect fraudulent transactions accurately
* Handle imbalanced datasets using SMOTE
* Reduce false positives and false negatives
* Improve overall financial transaction security

---

## Technologies Used
* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib, Seaborn
* imbalanced-learn

---

## Machine Learning Model
* Random Forest Classifier
* SMOTE (Synthetic Minority Oversampling Technique)

---

## Features
* Data preprocessing and cleaning
* Handling imbalanced data
* Feature importance analysis
* Confusion matrix visualization
* ROC curve analysis

---

## Dataset
* Transaction amount
* Transaction type
* Sender and receiver balances
* Fraud label

---

## Results
